四个原型页面入口

上传整个 输出 html 目录后，默认建议从“今日模块”开始浏览；四个页面现在已经互相打通，也可以从这里任选入口。

01

今日模块

查看首页数据、状态卡片与今日核心内容展示。

文件：01-今日模块.html 进入页面
02

关心模块

进入关心页，查看用户关系、提醒与互动内容布局。

文件：02-关心模块.html 进入页面
03

个人中心

查看个人资料、设备信息、设置项和账户相关页面结构。

文件：03-个人中心.html 进入页面
04

练级模块

进入练级页，查看成长体系、任务卡片与挑战玩法内容。

文件：04-练级模块.html 进入页面
访问方式：如果你的 GitHub Pages 发布目录就是这个文件夹，那么首页地址就是 .../index.html 或目录根地址；如果你希望直接进入“今日模块”，也可以把仓库根目录的跳转页指向 01-今日模块.html。
