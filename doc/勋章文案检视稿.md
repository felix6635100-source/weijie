# 勋章墙 · 文案统一检视稿

> 用途：勋章墙「我的勋章」页面分类与描述统一文案，对照《勋章获取规则.md》整理。
> 整体分两大 Tab：**成长历程** + **限时挑战**。
> 成长历程 Tab 下含 5 大类、20 个系列；每个系列固定 5 级（铁/铜/银/金/钻石）。

---

## Tab 1 · 成长历程

### 1. 新手引导

> **类型描述**：刚刚开始使用唯戒戒指的成长足迹，记录你与戒指的相识与磨合。
> **副文（短）**：戒指连接与佩戴的入门里程碑。

| 系列 | 系列描述（一句话） |
|---|---|
| 戒指连接 | 持续佩戴戒指的天数里程碑 |

> 一期含 **1 个系列 × 5 级 = 5 枚**。

---

### 2. 身体指标

> **类型描述**：8 项核心健康指标的长期达标轨迹，反映你身体状态的稳定与进步。
> **副文（短）**：心理压力 / 身体恢复度 / 睡眠 / 步数 / 血氧 / 体温 / HRV / 静息心率，每项 5 级。

| 系列 | 系列描述（一句话） |
|---|---|
| 心理压力 | 保持低压状态的达标天数累积 |
| 身体恢复度 | 身体恢复度达到优秀的累积天数 |
| 睡眠 | 睡眠达标（时长 + 深睡）的累积天数 |
| 步数 | 单日步数达标的累积天数 |
| 血氧 | 血氧饱和度达标的累积天数 |
| 体温 | 体温维持健康区间的累积天数 |
| HRV（心率变异性） | HRV 处于良好区间的累积天数 |
| 静息心率 | 静息心率维持健康区间的累积天数 |

> 一期含 **8 个系列 × 5 级 = 40 枚**。

---

### 3. 今日挑战

> **类型描述**：当日 7 项习惯挑战的完成次数累积，看见每一次小小的坚持。
> **副文（短）**：走路 / 喝水 / 睡眠 / 消耗 / 吃药 / 心肺锻炼 / 站立 完成次数累积。
> **统一规则**：每个系列 5 级 = 累计完成 3 / 7 / 15 / 30 / 60 次（同一挑战每天最多算 1 次完成）。

| 系列 | 系列描述（一句话） |
|---|---|
| 步数挑战 | 单日步数达标的累计完成次数 |
| 喝水挑战 | 单日喝水目标的累计完成次数 |
| 睡眠挑战 | 单日睡眠目标的累计完成次数 |
| 消耗挑战 | 单日热量消耗目标的累计完成次数 |
| 吃药挑战 | 按时服药的累计完成次数 |
| 心肺锻炼挑战 | 心肺锻炼目标的累计完成次数 |
| 站立挑战 | 单日站立目标的累计完成次数（缓解久坐颈椎与腰背压力） |

> 一期含 **7 个系列 × 5 级 = 35 枚**。

---

### 4. 单项里程碑

> **类型描述**：在某项行为上达到值得纪念的数字，把"日积月累"具象成一个个里程碑。
> **副文（短）**：单次突破或长期累积达到关键数值的成就。

| 系列 | 系列描述（一句话） |
|---|---|
| 步行里程碑 | 单日 / 累计步数达到关键数值（1 万步 / 2 万步 / 10 万步 / 50 万步 / 100 万步） |
| 消耗里程碑 | 单日 / 累计消耗千卡达到关键数值（500 / 1000 千卡 / 1 万 / 5 万 / 10 万 千卡） |
| 站立里程碑 | 累计站立次数达到关键数值（50 / 200 / 500 / 1000 / 2000 次） |
| 心肺锻炼里程碑 | 单次 / 累计心肺锻炼时长达到关键数值（10 / 30 分钟 / 100 / 1000 / 10000 分钟） |
| 喝水里程碑 | 单日 / 累计喝水量达到关键毫升数（1 万 / 10 万 / 50 万 / 100 万 毫升） |
| 深度睡眠里程碑 | 单晚 / 累计深度睡眠时长达到关键数值（1 / 2 小时 / 10 / 50 / 100 小时） |

> 一期含 **6 个系列 × 5 级 = 30 枚**。
>
> **与"今日挑战"区别**：今日挑战看"次数累积"，强调坚持；单项里程碑看"绝对值突破"，强调高光时刻。

---

### 5. 三环成绩

> **类型描述**：每天三环结算获得的金/银/铜牌，按颜色独立累计的长期收藏。
> **副文（短）**：每天只能得 1 枚（金/银/铜三选一），按颜色累计升级。
> **原型对应**：截图里的「今日圆环」区块 — **建议改名为「三环成绩」或「三环徽章」**（与命名一致，避免和"今日圆环"混淆）。

| 系列 | 系列描述（一句话） |
|---|---|
| 金牌系列 | 累计获得金牌（当日三环全达标）的枚数里程碑（1 / 7 / 30 / 90 / 180 枚） |
| 银牌系列 | 累计获得银牌（当日三环中 2 环达标）的枚数里程碑（1 / 7 / 30 / 90 / 180 枚） |
| 铜牌系列 | 累计获得铜牌（当日三环中 1 环达标）的枚数里程碑（1 / 7 / 30 / 90 / 180 枚） |

> 一期含 **3 个系列 × 5 级 = 15 枚**。

## Tab 2 · 限时挑战

> Tab 副标题（候选）：「参与限时活动收获的纪念勋章」
> **统一材质**：所有限时挑战勋章均为「多彩勋章」，不使用铁/铜/银/金/钻石梯度。

| 类型 | 类型描述（一句话） | 示例 |
|---|---|---|
| 月度徽章 | 当月活动周期内完成指定目标 | 春日走路马拉松、秋日补水巡航、夏夜早睡计划 |
| 季度徽章 | 中周期主题运营 / 品牌联动 / KOL 联名 | 冬日恢复挑战、KOL 同步训练营 |
| 年度徽章 | 全年大型挑战 / 顶级荣誉展示 | 年度城市挑战者、年度限定收藏家 |

> 数量不固定，由运营按周期上新；活动结束后勋章保留在用户图鉴中。

## 现状对照（截图问题）

按你截图的勋章墙 → 成长历程 Tab 下的命名建议如下：

| 截图所示 | 建议改为 | 理由 |
|---|---|---|
| 「今日圆环」 0/15 枚 · 获取金牌、银牌、铜牌的累计次数 | **「三环成绩」** 0/15 枚 · 三环金/银/铜牌的累计收藏 | "今日圆环"易与今日 Tab 的"三环活动"混淆；它们的语义是"当天三环结算成绩"，叫"三环成绩"更直接 |
| （应该还有的其它分类项） | 按上面 5 大类完整露出 | 截图里只看到一个分类块，建议把 5 大类都做成手风琴或 Tab |

## 建议的 UI 信息层级

```
我的勋章
├─ Tab: 成长历程  (×125)
│   ├─ 新手引导        1 系列 · 5 枚
│   ├─ 身体指标        8 系列 · 40 枚
│   ├─ 今日挑战        7 系列 · 35 枚
│   ├─ 单项里程碑      6 系列 · 30 枚
│   └─ 三环成绩        3 系列 · 15 枚   ← 原"今日圆环"建议改名
│
└─ Tab: 限时挑战  (数量随运营)
    ├─ 月度徽章
    ├─ 季度徽章
    └─ 年度徽章
```

每个类型下点击可展开/折叠该类下的所有系列；每个系列展示 5 个等级（铁/铜/银/金/钻石）勋章网格。

## 用户视角文案建议（如果要直接给 C 端展示）

每个类型卡顶部的描述行，建议都用一句"用户感知"的话，不出现"达标""结算"等内部术语：

| 类型 | 给用户的副文（候选） |
|---|---|
| 新手引导 | 与戒指相识的每一步，都值得记下 |
| 身体指标 | 8 项指标的长期表现，看见身体的进步 |
| 今日挑战 | 每天一点小坚持，攒成属于你的勋章 |
| 单项里程碑 | 数字越大，越值得纪念 |
| 三环成绩 | 三环结算的每一枚奖牌，都收藏在这里 |
| 限时挑战 | 参与活动获得的限定纪念 |
